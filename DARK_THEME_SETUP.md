# Настройка темной темы в LinguaFlow

Проект LinguaFlow теперь поддерживает полноценную темную тему с возможностью переключения между светлой, темной и системной темами.

## Компоненты темной темы

### 1. ThemeProvider (`src/components/theme-provider.tsx`)
- Контекст для управления темой
- Сохранение выбора пользователя в localStorage
- Поддержка системной темы

### 2. ThemeToggle (`src/components/theme-toggle.tsx`)
- Компонент переключателя темы
- Dropdown меню с тремя опциями: Светлая, Темная, Системная
- Анимированные иконки солнца и луны

### 3. DropdownMenu (`src/components/ui/dropdown-menu.tsx`)
- UI компонент для выпадающего меню
- Основан на Radix UI
- Полная поддержка темной темы

## Настройки CSS

### Переменные темы (`src/app/globals.css`)
```css
:root {
  /* Светлая тема */
  --background: 0 0% 100%;
  --foreground: 222.2 84% 4.9%;
  /* ... другие переменные */
}

.dark {
  /* Темная тема */
  --background: 222.2 84% 4.9%;
  --foreground: 210 40% 98%;
  /* ... другие переменные */
}
```

### Градиентные фоны
- Светлая тема: `linear-gradient(to bottom right, rgb(238 242 255), rgb(255 255 255), rgb(207 250 254))`
- Темная тема: `linear-gradient(to bottom right, rgb(17 24 39), rgb(31 41 55), rgb(17 24 39))`

## Интеграция в Layout

### Root Layout (`src/app/layout.tsx`)
```tsx
<ThemeProvider
  defaultTheme="dark"
  storageKey="linguaflow-ui-theme"
>
  {children}
</ThemeProvider>
```

## Использование в компонентах

### Добавление переключателя темы
```tsx
import { ThemeToggle } from '@/components/theme-toggle'

// В компоненте
<ThemeToggle />
```

### Использование хука темы
```tsx
import { useTheme } from '@/components/theme-provider'

const { theme, setTheme } = useTheme()
```

## Tailwind CSS классы

Все компоненты автоматически поддерживают темную тему через:
- `dark:` префикс для темных стилей
- CSS переменные для цветов
- Автоматическое переключение через класс `.dark`

✅ Главная страница (`src/app/page.tsx`)
✅ Страница уроков (`src/app/lessons/page.tsx`)

## Настройки по умолчанию

- **Тема по умолчанию**: Темная (`defaultTheme="dark"`)
- **Ключ хранения**: `linguaflow-ui-theme`
- **Поддержка системной темы**: Да

## Особенности

1. **Плавные переходы**: Все элементы имеют анимированные переходы между темами
2. **Сохранение выбора**: Выбор темы сохраняется в localStorage
3. **Системная тема**: Автоматическое определение системных настроек
4. **Адаптивность**: Все компоненты корректно отображаются в обеих темах
5. **Accessibility**: Поддержка screen readers и клавиатурной навигации

## Кастомизация

Для изменения цветов темной темы отредактируйте переменные в `src/app/globals.css`:

```css
.dark {
  --primary: 217.2 91.2% 59.8%; /* Основной цвет */
  --background: 222.2 84% 4.9%; /* Фон */
  --foreground: 210 40% 98%; /* Текст */
  /* ... другие переменные */
}
```
