# LinguaFlow Database Schema

## Обзор

База данных LinguaFlow содержит 21 таблицу, полностью покрывающую все функциональные требования из PRD. Все таблицы защищены Row Level Security (RLS) политиками.

## Основные таблицы

### Пользователи и аутентификация
- **profiles** - Профили пользователей (расширение auth.users)
- **user_settings** - Настройки пользователя
- **user_streaks** - Отслеживание серий обучения
- **notifications** - Уведомления пользователей

### Учебный контент
- **languages** - Языки (English, German)
- **courses** - Курсы по уровням (A1, A2, B1, B2)
- **themes** - Темы внутри курсов
- **lessons** - Уроки с интерактивными слайдами
- **vocabulary** - Словарь с переводами и транскрипциями
- **lesson_vocabulary** - Связь уроков со словами
- **audio_files** - Аудиофайлы для произношения

### Прогресс и аналитика
- **lesson_progress** - Прогресс прохождения уроков
- **learning_sessions** - Сессии обучения для отслеживания времени
- **user_answers** - Детальные ответы пользователей
- **user_statistics** - Ежедневная статистика
- **user_lesson_preferences** - Предпочтения по урокам

### Карточки и SRS
- **flashcard_sets** - Наборы карточек
- **flashcards** - Индивидуальные карточки с SRS данными
- **flashcard_reviews** - История повторений карточек

### Геймификация
- **achievements** - Определения достижений
- **user_achievements** - Полученные пользователем достижения

## Ключевые функции

### Пользовательские функции
- `start_learning_session()` - Начать сессию обучения
- `end_learning_session()` - Завершить сессию с подсчетом статистики
- `record_user_answer()` - Записать ответ пользователя
- `get_user_analytics()` - Получить аналитику пользователя

### Система достижений
- `award_xp()` - Начислить опыт пользователю
- `check_and_award_achievements()` - Проверить и выдать достижения
- `create_notification()` - Создать уведомление

### SRS система
- `get_due_flashcards()` - Получить карточки для повторения
- `update_flashcard_review()` - Обновить карточку после повторения

### Утилиты
- `calculate_user_level()` - Рассчитать уровень по опыту
- `update_user_streak()` - Обновить серию обучения

## Безопасность

Все таблицы защищены RLS политиками:
- Пользователи видят только свои данные
- Публичный контент (уроки, словарь) доступен всем
- Админы имеют расширенный доступ для управления
- Системные функции могут обновлять статистику

## Триггеры

- Автоматическое создание профиля при регистрации
- Автоматическое создание настроек и серий для новых пользователей
- Автоматический пересчет уровня при изменении опыта
- Автоматическое обновление timestamp полей

## Индексы

Созданы индексы для оптимизации производительности:
- По пользователям для всех связанных таблиц
- По датам для статистики и сессий
- По времени повторения для карточек
- По статусу для уведомлений

## Миграции

Применено 8 миграций:
1. `initial_schema` - Основная схема
2. `rls_policies` - RLS политики
3. `database_functions` - Базовые функции
4. `seed_data` - Начальные данные
5. `additional_tables` - Дополнительные таблицы
6. `additional_rls_policies` - RLS для новых таблиц
7. `additional_functions` - Дополнительные функции
8. `additional_triggers` - Триггеры для новых таблиц

База данных готова для полнофункционального приложения LinguaFlow!
