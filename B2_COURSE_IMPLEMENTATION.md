# Реализация страниц курсов B2 уровня

Реализованы страницы курсов B2 уровня для английского и немецкого языков в соответствии с дизайном из предоставленного изображения.

## Созданные файлы

### 1. English B2 (`src/app/lessons/english/b2/page.tsx`)
- Темная тема с градиентом `from-slate-900 via-blue-900 to-slate-900`
- 4 основных модуля курса
- Полная интеграция с системой темной темы

### 2. German B2 (`src/app/lessons/german/b2/page.tsx`)
- Темная тема с градиентом `from-slate-900 via-red-900 to-yellow-900`
- 4 специализированных модуля для немецкого языка
- Адаптированный контент для немецкоязычной аудитории

## Особенности реализации

### Дизайн и UI
- **Темная тема по умолчанию**: Страницы используют темную цветовую схему
- **Градиентные фоны**: Красивые градиенты для создания атмосферы
- **Карточки модулей**: Интерактивные карточки с hover-эффектами
- **Адаптивная верстка**: Полная поддержка мобильных устройств

### Модули курсов

#### English B2
1. **Advanced Grammar** - Продвинутые грамматические структуры
2. **Business English** - Деловой английский язык
3. **Academic English** - Академический английский
4. **Advanced Communication** - Продвинутое общение

#### German B2
1. **Politik und Gesellschaft** - Политика и общество
2. **Medien und Technologie** - Медиа и технологии
3. **Kultur und Kunst** - Культура и искусство
4. **Wissenschaft und Umwelt** - Наука и экология

### Компоненты и функциональность

#### Header
- Логотип LinguaFlow
- Переключатель темы (ThemeToggle)
- Кнопка "Назад к урокам"

#### Основной контент
- **Заголовок курса**: Флаг страны + название + описание
- **Сетка модулей**: 2x2 сетка с карточками модулей
- **Информационные блоки**: "Что вы изучите" и "Уровень B2 означает"
- **Статистика курса**: 4 карточки со статистикой
- **CTA секция**: Призыв к действию с кнопками

#### Интерактивные элементы
- **Hover-эффекты**: Карточки поднимаются при наведении
- **Анимации**: Плавные переходы и трансформации
- **Иконки**: Lucide React иконки для каждого модуля
- **Градиенты**: Уникальные цветовые схемы для каждого модуля

### Цветовые схемы модулей

#### English B2
- **Advanced Grammar**: Purple (`from-purple-500 to-purple-600`)
- **Business English**: Green (`from-green-500 to-green-600`)
- **Academic English**: Blue (`from-blue-500 to-blue-600`)
- **Advanced Communication**: Orange (`from-orange-500 to-orange-600`)

#### German B2
- **Politik und Gesellschaft**: Red (`from-red-500 to-red-600`)
- **Medien und Technologie**: Blue (`from-blue-500 to-blue-600`)
- **Kultur und Kunst**: Purple (`from-purple-500 to-purple-600`)
- **Wissenschaft und Umwelt**: Green (`from-green-500 to-green-600`)

### Статус модулей
- Все модули помечены как "Скоро доступно"
- Кнопки отключены (`disabled`)
- Ссылки ведут на соответствующие страницы модулей

### Адаптивность
- **Мобильные устройства**: Сетка модулей становится одноколоночной
- **Планшеты**: Двухколоночная сетка
- **Десктоп**: Полная двухколоночная сетка с оптимальными отступами

### Интеграция с темной темой
- Полная поддержка переключения тем
- Адаптивные цвета для светлой и темной тем
- Правильные контрасты для читаемости
- Плавные переходы между темами

## Технические детали

### Используемые компоненты
- `Card`, `CardContent`, `CardHeader`, `CardTitle` - для карточек
- `Badge` - для статусов и меток
- `Button` - для интерактивных элементов
- `ThemeToggle` - для переключения темы
- Иконки из `lucide-react`

### Стилизация
- Tailwind CSS классы
- Кастомные градиенты
- Backdrop blur эффекты
- Анимации и переходы

### Навигация
- Интеграция с Next.js Link
- Правильная структура URL
- Breadcrumb навигация
