## 🎯 Обзор проекта

**LinguaFlow** - это современное веб-приложение для изучения языков с интерактивными уроками, системой карточек и геймификацией.

### Ключевые особенности

- 🎓 **Интерактивные уроки** с различными типами слайдов
- 🃏 **Система карточек** с алгоритмом интервального повторения (SRS)
- 🏆 **Геймификация** с достижениями и системой опыта
- 📊 **Аналитика прогресса** с детальной статистикой
- 🌙 **Темная/светлая тема** с автоматическим переключением
- 📱 **Адаптивный дизайн** для всех устройств
- 🔐 **Безопасная аутентификация** через Supabase

### Поддерживаемые языки

- 🇬🇧 Английский (A1-C1)
- 🇩🇪 Немецкий (A1-B2)

## 🏗️ Архитектура

### Технологический стек

**Frontend:**
- Next.js 16+ (App Router)
- React 19+
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zustand (управление состоянием)
- Recharts (графики)

**Backend:**
- Supabase (PostgreSQL)
- Supabase Auth
- Supabase Storage
- Row Level Security (RLS)

**Инструменты разработки:**
- Bun (пакетный менеджер)
- ESLint
- Jest (тестирование)
- Vercel (деплой)

### Архитектурные принципы

1. **Компонентный подход** - переиспользуемые UI компоненты
2. **Server-Side Rendering** - оптимизация производительности
3. **Type Safety** - полная типизация с TypeScript
4. **Security First** - RLS политики на уровне БД
5. **Mobile First** - адаптивный дизайн

## 🚀 Установка и настройка

### Предварительные требования

- Node.js 18+
- Bun 1.0+
- Аккаунт Supabase

### Локальная установка

1. **Клонирование репозитория:**
```bash
git clone https://github.com/your-username/linguaflow.git
cd linguaflow
```

2. **Установка зависимостей:**
```bash
bun install
```

3. **Настройка переменных окружения:**
```bash
cp .env.example .env.local
```

Заполните `.env.local`:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_DEFAULT_KEY=your_supabase_anon_key
```

4. **Запуск в режиме разработки:**
```bash
bun dev
```

Приложение будет доступно по адресу: http://localhost:3000

### Настройка Supabase

1. Создайте новый проект в [Supabase](https://supabase.com)
2. Выполните миграции из папки `supabase/migrations/`
3. Настройте RLS политики
4. Загрузите начальные данные

## 🎯 Функциональность

### Реализованные функции

#### 🔐 Аутентификация и профиль
- Регистрация и вход через Supabase Auth
- Профиль пользователя с аватаром и статистикой
- Система уровней и опыта (XP)
- Персональная панель управления

#### 📚 Система обучения
- **Иерархия контента**: Язык → Курс → Тема → Урок
- **Поддерживаемые языки**: Английский, Немецкий
- **Уровни**: A1 (начальный), B2 (продвинутый)
- **Интерактивные уроки** с различными типами слайдов:
  - `theory` - Теоретический материал
  - `quiz_single` - Одиночный выбор
  - `quiz_multiple` - Множественный выбор
  - `fill_gap` - Заполнение пропусков
  - `match` - Сопоставление элементов

#### 🃏 Система карточек (Flashcards)
- Автоматическое создание карточек из изученной лексики
- Алгоритм интервального повторения (SRS)
- Различные режимы повторения
- Отслеживание прогресса запоминания

#### 🏆 Геймификация
- Система достижений с уникальными карточками

- Отслеживание серий обучения

#### 📊 Аналитика и статистика
- Детальная статистика прогресса
- Графики активности по дням/неделям
- Отслеживание времени обучения

#### 🔍 Поиск и навигация
- Глобальный поиск по урокам и контенту
- Быстрая навигация (Cmd/Ctrl + K)
- Фильтрация по уровням и темам
- Поиск упражнений

#### 🎨 Пользовательский интерфейс
- Темная и светлая темы
- Адаптивный дизайн (mobile-first)

## 🗄️ API и база данных

- **profiles** - Профили пользователей
- **languages** - Поддерживаемые языки
- **courses** - Курсы по уровням
- **themes** - Темы курсов
- **lessons** - Интерактивные уроки
- **vocabulary** - Словарь
- **flashcards** - Система карточек
- **achievements** - Достижения
- **user_statistics** - Статистика пользователей

### API эндпоинты

Все API запросы проходят через Supabase клиент с автоматической аутентификацией.

**Основные операции:**
- `GET /api/lessons` - Получить список уроков
- `POST /api/lessons/progress` - Сохранить прогресс урока
- `GET /api/flashcards` - Получить карточки для повторения
- `POST /api/flashcards/review` - Отправить результат повторения
- `GET /api/statistics` - Получить статистику пользователя
