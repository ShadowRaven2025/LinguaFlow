# LinguaFlow - Фундаментальный документ

## Цель проекта
Создание современного веб-приложения для изучения языков с интерактивными уроками, системой карточек и геймификацией.

## Рамки проекта
- Веб-приложение на Next.js 16+
- Поддержка английского (A1-C1) и немецкого (A1-B2) языков
- Аутентификация через Supabase
- Деплой на Vercel

## Текущий статус
- Основной функционал реализован
- Активная разработка уроков
- Деплой на Vercel

## Project Deliverables
- **AUTH**: Аутентификация и профиль пользователя - 15%
- **LESSONS**: Система обучения (уроки, курсы, темы) - 20%
- **FLASHCARDS**: Система карточек с интервальным повторением - 15%
- **GAMIFICATION**: Геймификация и система достижений - 15%
- **ANALYTICS**: Аналитика и статистика прогресса - 10%
- **SEARCH**: Глобальный поиск и навигация - 10%
- **UI**: Пользовательский интерфейс (темы, адаптивность) - 15%
