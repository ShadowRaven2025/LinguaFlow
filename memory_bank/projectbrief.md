# LinguaFlow - Фундаментальный документ

## Цель проекта
Создание современного веб-приложения для изучения языков с интерактивными уроками, системой карточек и геймификацией.

## Рамки проекта
- Веб-приложение на Next.js 16+
- Поддержка английского (A1-C1) и немецкого (A1-B2) языков
- Аутентификация через Supabase
- Деплой на Vercel

## Текущий статус
- Основной функционал реализован
- Активная разработка уроков
